# Downloadable Resources

This folder contains downloadable resources like PDFs, templates, and checklists that are featured on the Resources page.

## File Organization

- **PDFs**: Guides, frameworks, and comprehensive resources
- **Templates**: Editable documents, spreadsheets, and design files
- **Checklists**: Step-by-step guides and audit tools

## Adding New Resources

1. Add your file to the appropriate subfolder
2. Update the `downloadableResources` array in `src/pages/ResourcesPage.tsx`
3. Ensure the file is accessible via the download URL

## File Types Supported

- PDF documents
- Word documents (.docx)
- Excel spreadsheets (.xlsx)
- Google Docs/Sheets (via shareable links)
- Figma files (via shareable links)
- Notion pages (via shareable links)

## Naming Convention

- Use descriptive, SEO-friendly filenames
- Use lowercase letters and hyphens
- Include version numbers if applicable
- Examples: `product-story-framework-v2.pdf`, `voice-tone-template.docx`
